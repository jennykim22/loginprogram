# loginprogram
a simple bulletin board web application integrated with a local LLM-based AI afent that supports document search, collection, highlighting, and summarizaiton

## Features
- User-friendly bulletin board system
- Local LLM-based AI agent integration
- Document search (semantic search supported)
- Automated document collection
- Text highlighting of relevant content
- AI-powered summarization

## Tech Stack
- Backend: python FastAPI
- Frontend: HTML-CSS-JS
- AI Model: ollama qwen2.5
- Database: postgreSQL
